Initial Thoughts
----------------

* Keyboard or UI controls to add/remove turtles
* Each turtle has own name, appearance, draw color, other properties
* Initially all on the same page, but separate pages should be possible down the line
* The ability to select turtles would be needed
* Center turtle would have to be rehomed with the turtle controls (from the page form) - you could add an 'average' for the page I suppose
* Page rotation would probably have to belong to the turtle as well - again an 'average' could be doable, might be strange though
* Consider turning the turtles into web-components??

Early experiments with focus for the turtle have been mixed.
