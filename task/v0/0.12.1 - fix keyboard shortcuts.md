Affected
--------

* A prevent default is affecting ctrl-shift-r
* ctrl-c for copy clears the drawing instead
* Browser controls for full-page zoom: ctrl+ & ctrl-
* ctrl-shift-t


Fix
---
I've quickly added this into the key listener to filter out `alt` `ctrl` & `meta` combos:
```js
	if (!event.altKey && !event.ctrlKey && !event.metaKey) {
		...
	}
```
