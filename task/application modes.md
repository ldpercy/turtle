Have occasionally wondered about the idea of having different app modes, for example simple and advanced, depending on the intended audience.
Eg you could turn off the polar grid and radians for younger users, but leave them in for more advanced users.

Also currently wondering about a 'setup' mode where the user can specify things like the space and turtle(s) settings.
Whether that's done in turtle-cmd syntax or just with ui settings, not sure yet.
